## 为什么要申请 GitHub 学生包？

GitHub Student Developer Pack 被称为“学生开发者的军火库”，里面包含了上百种付费工具的免费使用权。对我来说，最核心的福利是：

1.  **DigitalOcean 云服务器：** 送了 **$200 的信用额度**（有效期1年）。我现在部署博客和问卷系统的这台**新加坡服务器**，就是用这个额度兑换的，相当于免费白嫖一年！
2.  **JetBrains 全家桶：** IntelliJ IDEA、WebStorm 的专业版免费用（一年能省几百刀）。
3.  **Namecheap 域名：** 免费送一个 `.me` 结尾的域名（比如我的 `linjingbiao.me`），还有免费的 SSL 证书。
4.  **Microsoft Azure：** 也是送服务器额度，不用信用卡验证。

## 申请避坑指南（关键！）

申请地址：[https://education.github.com/pack](https://education.github.com/pack)

看似简单，但很多人（包括之前的我）都挂在了验证上。以下是我这次**“一次过”**的血泪经验：

### 1. 准备材料

* **手写的申请报告**（英文版最好），这个可以去参考抖音或者b站的手写内容。
* **学校邮箱（edu.cn）：** 如果有，一定要绑定到 GitHub 账号上，这是通过率最高的凭证。
* **GitHub 个人资料：** 把你的 Bio（简介）改得像个正经学生，比如 "Student at XXX University, graduating in 2026".

### 2. 也是最重要的一点：关掉代理！

**这是我之前失败的主要原因。** GitHub 会检测你的 IP 地址是否在学校附近。

* 如果你挂着梯子（比如显示在美国），但你选的学校在中国，**秒拒**。
* **正确姿势：** 手机连学校 WiFi 或流量，使用**手机摄像头**实时拍照上传证件（不要上传相册里的老照片）。
* **把你学校周边的经纬度查出来：** 这个参考网上教程，查出来后点击F12修改定位地址

### 3. 写申请理由（小作文）

不用写太长，但要真诚。我大概是这么写的：

> "I am a senior student majoring in Computer Science (graduating June 2026). I am currently building a full-stack survey system using Vue and Spring Boot. I need the DigitalOcean credits to deploy my project and the JetBrains license for development. This pack will greatly help my career preparation."
